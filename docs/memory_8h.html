int  page_count 
)

Allocate a number of system pages using mmap()

If page_count is <= 0, this method will fail.

The size of the allocated region, in bytes, will be main_context.system_page_size * page_count

Returns
A pointer to the allocted region on success, or 0 on failure
Parameters
[in]main_contextThe main context
[in]page_countThe number of pages to allocate

Reallocate a region allocated by memory_page_alloc()

Uses mremap() on non-BSD/Apple systems, otherwise just munmap() and mmap().

NOTE: Expect the re-allocated region to start at a different address in memory than offset

Returns
A pointer to the re-allocated region on success, or 0 on failure
Parameters
[in]main_contextThe main context
[in]offsetA pointer to the start of the region to reallocate
[in]old_page_countThe old page count
[in]page_countThe new page count

int  new_amount

Reallocate a region allocated using memory_alloc()

If new_amount is greater than old_amount, this method will initialize the bytes that are added to 0 by using memset().

Returns
A pointer to the re-allocated region on success, or 0 on failure
Parameters
[in]regionA pointer to the region that was allocated using memory_alloc()
[in]old_amountThe current size of the region in bytes
[in]new_amountThe size that the region should change to in bytes
